# AccessOS

## Overview
Browser-based assistive system for motor disabilities.

## Features
- Eye tracking (paralysis support)
- Tremor filtering
- Face gesture control

## Tech Stack
- React (Frontend)
- WebGazer.js
- MediaPipe

## Project Structure
/frontend → UI & features  
/backend → future APIs  
/docs → documentation  
/assets → demo files  

## Setup
cd frontend  
npm install  
npm start
